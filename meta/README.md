# meta — experiments on this repository

The rest of this repository runs experiments on learning systems. This directory runs
one on *the repository itself*, under the same rules: a gesture, a signature, a kill
criterion, written down first.

It is kept apart from [`story/`](../story/), [`paper/`](../paper/) and
[`repro/`](../repro/) on purpose. Nothing here bears on the invariant. If you came for
the science, none of this is it — go to [chapter 1](../story/ch01-the-graveyard-of-resemblances.md)
or the [paper](../paper/draft.md).

| file | what it is |
|---|---|
| [`discovery-preregistration.md`](discovery-preregistration.md) | the reception experiment, registered before the first reader |
| [`traffic_snapshot.py`](traffic_snapshot.py) | the instrument: keeps GitHub's 14-day traffic window from evaporating |
| `traffic/ledger.json` | the record it writes (appears after the first workflow run) |
